🛠️ Quick Setup Instructions

  1. Download: Click the download button above
  2. Extract: Unzip the downloaded file
  3. Install Python: Make sure Python 3.8+ is installed
  4. Install Dependencies: 
    pip install -r requirements.txt
  5. Install Playwright Browsers: 
    playwright install chromium
  6. Run the Application: 
    python app.py
  7. Open Browser: Go to http://localhost:5000
  8. Start Scraping: Enter a username and watch the magic happen!

🔧 System Requirements

  • Operating System: Windows, macOS, or Linux
  • Python: Version 3.8 or higher
  • RAM: At least 4GB (8GB recommended for large profiles)
  • Storage: 500MB free space for dependencies
  • Internet: Stable connection required
  • Browser: Chromium will be auto-installed by Playwright

❓ Why Local Installation?

This Twitter scraper uses advanced browser automation that requires:

  • Real Browser: Playwright launches actual Chromium browser
  • User Login: You need to log in to Twitter for full access
  • File System: Saves tweets to your local computer
  • Long Running: Can take minutes to scrape large profiles
  • System Resources: Needs more memory/CPU than serverless allows

These features make it impossible to run on static hosting platforms like Netlify, but provide much more powerful functionality than simple web scrapers.
